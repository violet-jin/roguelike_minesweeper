在推荐方向中选择了经典游戏复刻，一开始在考虑记忆卡牌的方向，但好像没什么意思，后来想到了"扫雷"，先定下了游戏根本。再思考了改编的方向，改编不太好做，扫雷已经是很经典且完善的小游戏，于是想着结合一些新的形式，比如说肉鸽，做成冒险者探险的形式。目前游戏还不是很完善，在每层的玩法上基本上达到要求，但层与层之间还想设置守关boss，这样才能把攒的钱利用起来。总之，还得继续调试。
不过，在搜索肉鸽扫雷的英文时，发现steam上好像有这个名字的游戏……
